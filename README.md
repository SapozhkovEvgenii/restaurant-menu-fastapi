Application launch:
 - clone the repository
 - go to project folder

    ```
    cd restaurant-menu-fastapi/
    ```
 - running app:

    ```
    docker-compose up -d
    ```
 - running tests:

    ```
    docker-compose -f docker-compose-tests.yaml up
    ```
